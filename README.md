A **modern, enterprise-grade** data reconciliation system designed for legal professionals to efficiently match case records with the Kenya Revenue Authority (KRA) iLaw database. Featuring an intuitive interface, AI-powered assistance, and advanced matching algorithms.

![Dashboard Preview](https://via.placeholder.com/800x400?text=Dashboard+Preview)

## ✨ Features

### 🎯 Core Functionality
- **Bulk Reconciliation**: Process hundreds of cases simultaneously
- **Multi-Strategy Matching**: Combines token-based and string-based fuzzy matching
- **Court-Type Filtering**: Automatically identifies and filters by court type (TAT, HC, CA, SU)
- **Confidence Scoring**: Calculates match confidence percentage for each case

### 🤖 AI Assistant (KRA)
- **Natural Language Interface**: Ask questions about your data in plain English
- **Intelligent Analysis**: Get insights and summaries automatically
- **Context-Aware**: Understands your current data and provides relevant suggestions
- **24/7 Availability**: Instant answers to common questions

### 📊 Modern Dashboard
- **Real-time Metrics**: Live updates on reconciliation status
- **Interactive Charts**: Visualize data with Plotly graphs
- **Color-Coded Results**: Instant visual identification of case status
- **Export Options**: Generate Excel, CSV, and PDF reports

### 🔒 Enterprise Features
- **Secure Authentication**: Credential management with environment variables
- **Session Management**: Automatic session renewal for KRA iLaw
- **Audit Logging**: Comprehensive logging of all operations
- **Error Handling**: Graceful error recovery with detailed logs
